# OrigamiPost v2 ドキュメント

Three.js を使用して折り紙の折り動作を再現するコンポーネント。ユーザーが頂点をドラッグ&ドロップすると折り線を計算し、板を分割・回転して繰り返し折ることができる。折り手順の履歴（`FoldStep` の列）を唯一の状態源とし、現在の形状は常に初期状態からのリプレイで導出する、という設計が全体を貫く原則になっている。

パスの表記はすべて `src/components/v2/OrigamiPost/` からの相対パス。各 `utils/` にはテスト（`index.test.ts`）が併設されており、仕様の実例集として読める。

## 読む順序

- はじめて読む場合は **01 → 02** を先に読む。以降のドキュメントはこの2つを前提にしている
- ユーザー操作の流れを追いたい場合は **03 → 04 → 05** の順で、1回の折りが確定するまでを時系列に追える
- **06〜08** はそれぞれ独立して読める

## 既知の制限（現在の仕様）

- 埋もれた板を他の板を貫通して折れてしまう判定はしない（[06 破れ防止](./06-tear-prevention.md) の展開図連結性データは、将来この判定を実装するための基礎になる）
- 折り角は 180 度固定
